# Vehicles

## Get list of Vehicles

### Request

`GET /api/vehicles`

### Response

List of all vehicles

## Create a new Vehicle

### Request

`POST /api/vehicles`

### Response

Posted vehicle

## Get a specific Vehicle

### Request

`GET /api/vehicles/id`

### Response

Requested vehicle

## Patch a vehicle

### Request

`PATCH /api/vehicles/:id/`

### Response

Patched vehicle

## Delete a Vehicle

### Request

`DELETE /api/vehicles/id`

### Response

Deleted vehicle
